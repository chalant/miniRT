## Requirements
- **libbsd-dev**: Required for building the project on Linux

## Installation
### On Linux
1. **Clone repository:**
```
git clone https://github.com/chalant/miniRT.git
```
2. **Install dependencies:**
```
sudo apt-get install libbsd-dev
```
3. **Build project:**
```
make
```
### On Mac OS

1. **Clone repository:**
```
git clone https://github.com/chalant/miniRT.git
```
2. **Build project:**
```
make
```
## Usage
```
./minirt [file-path]
```
## Example
```
./minirt map.rt
```
